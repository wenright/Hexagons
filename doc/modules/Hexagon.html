Initialize a new hexagon object Note that coordinates are 3 dimensional, but will be drawn on a 2D plane

  • x number x-coordinate to draw hexagon
  • y number y-coordinate to draw hexagon

move (dir, remove)
Move a hexagon in a certain direction (using a tween animation)

Parameters:

Set the world coordinates for a hexagon, and tween to that loccation

Parameters:

  • x number x-coordinate to move hexagon to
  • y number y-coordinate to move hexagon to
  • z number z-coordinate to move hexagon to
  • margin number How far apart the hexagons should be
  • remove boolean Wether or not hexagon should be removed when finished moving

slideHexagons (axis, axisValue, dir, inverted)
Slide hexagons in a certain direction along an axis

Parameters:

  • axis string The axis to slide hexagon along. Ex 'y'
  • axisValue number This is the x, y, or z coordinate of the row to be moved
  • dir string The compass direction that the hexagons will slide
  • inverted boolean Used to determine sorting order so that hexagons slide in the right order
