scripts.utils Functions

math.between (x, first, second)
Check if a value is between two others. Parameters need not be in ascending order.

Parameters:

  • x number The value to compare with the other two
  • first number The first value (could be lower or upper bound)
  • second number The second value (could be lower or upper bound)
math.sign (x)
Returns the sign of a given number

Parameters:

  • x number The number whose sign we will check

Returns:

    number -1 if x < 0, 1 if x > 0, or 0 if x is 0

states.game Functions

Game
Holds current game information

Fields:

  • gridRadius How many hexagons should be drawn
  • hexSize The draw size for a hexagon
  • pointerStart Where the user has first clicked the screen
  • canMove Determines if the user can slide. False while tween animation is playing
  • started True once the initial tweenIn animation has finished
  • over True when the player has won the game
  • slideDirection The direction the player has started sliding
  • slideAxis The axis the player has started sliding on
  • isDragged True if the player is currently clicking the screen
  • stencilFunction Used to prevent certain areas from drawing
Game:init ()
Initialize the game
Game:update (dt)
Called once per frame, updates the game

Parameters:

  • dt number Time passed between frame draws

Draw the current frame
Game:over ()
Called when the player completes a hexagon and the game is over
